## El Reto

Crear una experiencia VR que fuera más allá del entretenimiento superficial, ofreciendo puzzles técnicos que requirieran pensamiento crítico y manipulación física realista de objetos.

## La Solución

Usando Unity con Hurricane VR para las interacciones físicas, construí una nave espacial detallada donde cada sistema (soporte vital, navegación, comunicaciones) puede fallar y requiere procedimientos específicos de reparación. El jugador debe diagnosticar problemas, encontrar herramientas y ejecutar reparaciones bajo presión de tiempo.

## Detalles Técnicos

- **Motor:** Unity con pipeline de renderizado optimizado para VR
- **Interacciones:** Hurricane VR para manipulación física de objetos
- **Puzzles:** Sistema modular de crisis con múltiples soluciones posibles
- **Audio:** Diseño sonoro espacial 3D para inmersión completa
