## El Reto

El negocio vendía principalmente por redes sociales y necesitaba un sitio web profesional que mostrara su catálogo de forma atractiva y permitiera recibir pedidos directamente, sin necesidad de infraestructura e-commerce compleja.

## La Solución

Elegí Astro para generar un sitio completamente estático con carga instantánea. El diseño usa gradientes y colores inspirados en filamentos de impresión 3D multicolor. El catálogo es filtrable por categorías y cada producto tiene formulario de pedido directo.

## Detalles Técnicos

- **Framework:** Astro con generación estática (0 JavaScript al cliente)
- **Styling:** Tailwind CSS con sistema de colores custom
- **Formularios:** FormSubmit para pedidos sin backend
- **Tipado:** TypeScript para type-safety completa
- **Deploy:** Hosting estático con CDN global
